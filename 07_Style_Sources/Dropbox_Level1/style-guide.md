# Dropbox Level 1 Style Guide

This guide captures the worksheet style implied by the Dropbox Level 1 folder.

## Overall Feel

- Teacher-made, practical, print-first worksheets.
- More visual and activity-driven than official EOW worksheets.
- Short task blocks, often one clear grammar or vocabulary target per file.
- Filename language is direct and usable as a generator tag: `VOC`, `how many`, `old young`, `in on under`, `Do you`, `Does she`, `song`.

## Common File Sequence

For each unit, the Dropbox style tends to organize worksheets like this:

```text
Unit packet / overview
01 Vocabulary or unit context
02 Grammar or vocabulary extension
03 Target grammar / sentence pattern
04 Target grammar / reading-support skill
05 Song or review
06 Optional extension
```

## Layout Signals

From visible thumbnails and filenames:

- Use picture-heavy tasks.
- Include writing lines under or beside pictures.
- Use bordered worksheet sections, often with simple colored frames.
- Keep one task visible and uncluttered.
- Use playful but clear student-facing instructions.
- For Level 1, prefer recognition, matching, tracing/copying, short answers, and sentence frames.

## Activity Tags To Use In Generators

| Tag | Meaning |
|---|---|
| `VOC` | Vocabulary introduction or review. |
| `with color` | Combine target vocabulary with color adjectives. |
| `how many` | Counting and plural noun question practice. |
| `old young` | Adjective/antonym practice. |
| `this is my` | Possessive or self/family sentence frame. |
| `in on under` | Preposition practice. |
| `I'm wearing` | Present progressive clothing sentence frame. |
| `demon adj` | Demonstrative adjective practice. |
| `Do you` | Yes/No question practice. |
| `Does she` | Third-person Yes/No question practice. |
| `does doesnt` | Third-person simple present review. |
| `song` | Song/chant reinforcement worksheet. |
| `Bingo` | Vocabulary bingo game board. |
| `dictionary` | Picture dictionary / word-bank reference. |
| `Spiral Grammar` | Mixed-unit review. |

## Generator Instruction

When the user asks for a worksheet "like my Dropbox style", follow this priority:

1. Content scope from `01_EOW/`.
2. Reading/story alignment from `03_Reading_Base/`.
3. Grammar focus from `02_Grammar_Box/`.
4. Layout and activity sequence from this Dropbox style guide.

Do not copy a Dropbox worksheet exactly unless the user provides the file and asks for an edit. Use the style as a pattern.
