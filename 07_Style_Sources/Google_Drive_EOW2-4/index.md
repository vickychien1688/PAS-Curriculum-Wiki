# Google Drive EOW2-4 Style and Source Index

Google Drive contains official EOW material and worksheet folders for Levels 2-4.

These files are reliable for official scope and existing worksheet coverage. For local PAS worksheet feel, prefer Dropbox and Canva style references.

## Linked Source Map

See [Google Drive Source Map: Levels 2-4](../../00_Admin/google-drive-levels-2-4.md).

## Use Cases

Use Google Drive when you need:

- official EOW teacher manuals
- official worksheet files
- answer keys
- audio folders
- existing unit folder organization

Use Canva when you need:

- Reading Base editable designs
- Grammar Box editable designs
- Canva page size and layout references

Use Dropbox when you need:

- preferred PAS worksheet activity style
- locally adapted worksheets
- bingo, song, mini-book, and spiral grammar examples
