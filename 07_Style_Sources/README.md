# Style Sources

This folder stores style references that should influence worksheet layout, pacing, and task design.

Important distinction:

- EOW and Reading Base decide what content is allowed.
- Style Sources decide how the worksheet should feel and look.

Current sources:

- [Canva Source Index](Canva/index.md)
- [Dropbox Level 1 Style Index](Dropbox_Level1/index.md)
- [Dropbox Level 1 Style Guide](Dropbox_Level1/style-guide.md)
- [Google Drive EOW2-4 Index](Google_Drive_EOW2-4/index.md)

Source roles:

- Canva: Reading Base and Grammar Box master files.
- Dropbox: preferred worksheet style examples.
- Google Drive: official EOW files and worksheet references.
