# Canva Source Index

Canva is the editable source for Reading Base and Grammar Box.

It is also useful for locating existing tests, worksheet designs, and visual formats, but content scope still comes from EOW and the relevant curriculum source page.

## Main Canva Source Roles

| Role | Source |
|---|---|
| Reading Base master files | `r-level 1`, `r-level 2`, `r-level 3`, `r-level 4` folders |
| Grammar Box master files | `Gramma Box` folder and numbered level folders |
| EOW or test designs | Canva `EOW` folders and specific test designs |
| Visual layout reference | Existing PAS Canva worksheets and printable designs |

## Generator Rule

Canva references should guide:

- editable source location
- page size
- visual layout
- worksheet pacing
- final design reconstruction

They should not override EOW vocabulary, unit grammar, or reading source boundaries.
