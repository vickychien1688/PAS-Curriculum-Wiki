## Naming Notes

- Reading Base folders are named `r-level X` under the `The Reading Base` root folder.
- Individual designs may use `EOW X-UNIT Y READER`, `EOW X -UNITY Reader`, or older mixed naming.
- Levels 2-4 currently show Units 1-5 only in Canva; Units 6-8 readers are not yet present or not yet indexed.
- Some non-EOW reader/design files appear in search results. Confirm the containing folder before treating them as PAS source.

## Extraction Workflow

For each reader design:

```yaml
level:
unit:
canva_design:
canva_url:
page_count:
story_titles:
page_ranges:
target_vocabulary:
target_grammar:
worksheet_opportunities:
status:
```

Do not invent Reading Base story details when a Canva design has not been opened or exported.
