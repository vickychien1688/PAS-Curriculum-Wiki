# EOW4 Image Sheet Plan

Batches of **6 words** grouped by SAME CATEGORY so each generated sheet stays
style-consistent. Each sheet lists the six words, their target filenames
(`words/<word>/<word>-v1-color.png`), and a ready-to-paste ChatGPT prompt.

Workflow: paste the prompt into ChatGPT/DALL·E → get the 2×3 grid → slice into 6
icons → save each into its `words/<word>/` folder per `style-bible.md` naming, then
make the `bw` version. Mark each word `DONE` in `vocab-image-list-eow4.md`.

Face clause per category:
- objects / settings / abstract icons → `NO faces (objects)`
- animals / people / action figures → `cute friendly faces`

Total: **119 words → 20 sheets** (19 full + 1 with 5). Grouped: OBJECTS (sheets 1–7), ACTIONS (8–13), ADJECTIVE/ABSTRACT (14–18), PEOPLE (19), SETTINGS (20).
