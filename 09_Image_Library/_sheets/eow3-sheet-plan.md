# EOW3 Image Sheet Plan

Batches of **6 TODO words** grouped by **same category** (objects / people / actions /
settings / senses) so each generated sheet stays visually consistent. Every word here is
new (`TODO`); none reuse an existing folder.

Workflow per sheet:
1. Paste the prompt into ChatGPT (image tool) to get one 2×3 grid.
2. Crop the grid into 6 squares.
3. Save each into its target file `words/<word>/<word>-v1-color.png`, then make the `bw` version.
4. Mark each word `DONE` in `vocab-image-list-eow3.md`; paste the prompt into each `words/<word>/prompt.txt`.

Face clause: **objects / settings / senses → "NO faces (objects)"**; **people / animals → "cute friendly faces"**.
(Animals in EOW3: `goldfish`, `hamster`.)

Total: **113 words → 24 sheets**. To keep each grid one single category (objects/people/
actions/settings/senses) for style consistency, leftover words go on smaller grids
(2×3 of six where possible; otherwise 1×2, 1×3, 2×2, or single). Twelve sheets are full
2×3 grids of six; the other twelve are short same-category grids.

## Hard-to-depict words (flag for review)

These are abstract or easily confused as icons; check the cropped result reads at ~35 px:

- **protect**, **take-care-of-my-pet**, **help** — abstract caring actions; rely on clear two-figure poses.
- **come-home** — easily confused with "go out"; show a child stepping IN through a door.
- **compare** — abstract; shown as hands weighing two items.
- **price** vs **money** — keep distinct: `price` = a tag, `money` = coins + bill.
- **beautiful / delicious / terrible / ugly** — qualities, not things; use mood cues (sparkles, steam, wavy bad-smell lines) without faces.
- **loud / quiet** — opposite pair; differentiate by bold vs faint sound waves.
- **hard / soft** — opposite pair; keep on the same sheet for a consistent rock-vs-pillow contrast.
- **bitter / sour / salty / spicy / sweet** — all "taste"; rely on a recognisable example food each.
- **downhill / uphill** — mirror poses; keep them adjacent so the slope direction is the only difference.
- **fur** — a texture, not an object; render as a soft fur patch swatch.
