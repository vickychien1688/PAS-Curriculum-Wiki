# EOW2 Image Sheet Plan

Batch plan for generating all **102 new** EOW2 word icons (the 1 reuse, `chicken`, is skipped).
Each sheet = up to 6 icons of the **same category** so the style and the face rule stay consistent
within one generation. Generate the sheet, then split into single icons and save each as
`words/<word>/<word>-v1-color.png` per `style-bible.md` §4. (Produce the `bw` version after.)

Prompt format is fixed — paste exactly, only the face clause changes by category:
- Objects / settings / weather → `NO faces (objects)`
- Animals / people → `cute friendly faces`

Sheet order below groups by: **Animals → People (emotions) → Actions → Objects → Weather & Sky.**
