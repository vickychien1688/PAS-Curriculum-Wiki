# Vocabulary Image Master List — EOW1

Every EOW1 word that needs a picture. Build each prompt from `style-bible.md` §5,
using the "Pose seeds" below for the `{variant_pose}` slot. Generate `color` + `bw`
for every variant. Update **Status** as you go: `TODO` → `DONE`.

Legend — Type: `obj`=object, `per`=person, `act`=action, `adj`=adjective pair, `col`=color swatch, `set`=setting.
Variants = target number of pose variants.

- EOW1 words needing images: **94** (objects/people/settings) + adjective pairs + actions + 2 color swatches.
