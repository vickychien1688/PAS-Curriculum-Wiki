# Source Navigation

Single entry point for "where does this content live".

Read this page first when looking for source material or deciding where new material belongs.

## Platform Roles

| Platform | Owns | Never use it for |
|---|---|---|
| GitHub (this wiki) | Index, scope, mappings, prompts, specs. Single source of truth for *what is allowed*. | Storing editable design files. |
| Google Drive | Official EOW material: textbooks, teacher manuals, official worksheets, answer keys, audio. | Reading Base or Grammar Box editing. |
| Canva | Editable master files for Reading Base and Grammar Box, plus EOW test/worksheet designs. | Official scope decisions. |
| Dropbox | Preferred PAS worksheet style examples (the local look and feel). | Content scope decisions. |
| Local Downloads PDFs | Exports/backups for extraction and OCR only. | Treating as master source. |

## Conflict Rule

1. Scope conflicts: EOW official (Drive / extracted wiki pages) wins.
2. Reading/grammar text conflicts: Canva master wins over local PDF exports.
3. Style conflicts: Dropbox preferred style wins over official worksheet style.
4. Anything not found in the wiki: mark `DATA NEEDED`, do not invent.
