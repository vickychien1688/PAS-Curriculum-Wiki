## Generator Rule

When generating Level 2-4 worksheets:

1. Use the EOW teacher manual or unit folder for official scope.
2. Use official worksheet folders as reference, not as the preferred final style.
3. Use Canva Reading Base and Canva Grammar Box as the main reading/grammar sources.
4. Use Dropbox style sources when the user asks for the local PAS worksheet feel.
