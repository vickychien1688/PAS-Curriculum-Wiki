# EOW3 Unit 1: The World of Work

## Official Scope

| Field | Content |
|---|---|
| Theme | The World of Work |
| Goals | Talk about jobs; talk about where people work; say what I want to be. |
| Vocabulary 1 | bus driver, chef, doctor, farmer, firefighter, nurse, police officer, scientist, vet |
| Vocabulary 2 | artist, inventor, movie star, rock star, soccer player |
| Grammar 1 | Simple present for occupations (What does your father do? He's a chef. / Where does he work?) |
| Grammar 2 | Simple present with want + infinitive (What do you want to be one day? I want to be a movie star.) |
| Song / Sounds | Song: Work · /f/ as in farmer |
| Reading | Wonderful Work! |
| Value | Work hard. |
