# EOW3 Unit 2: Let's Eat!

## Official Scope

| Field | Content |
|---|---|
| Theme | Let's Eat! |
| Goals | Talk about foods; say what I like to eat; ask politely for things. |
| Vocabulary 1 | beans, bread, corn, ice cream, mango, meat, noodles, potato, tomato |
| Vocabulary 2 | cheese, chip, grape, nut, snack, yogurt |
| Grammar 1 | Any in questions and negative responses (Are there any tomatoes? No, there aren't any tomatoes.) |
| Grammar 2 | Polite requests with may (May I have some chips, please?) |
| Song / Sounds | Song: Let's Eat! · /m/ as in mango |
| Reading | Super Snacks! |
| Value | Eat good food. |
