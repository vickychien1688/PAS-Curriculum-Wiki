# EOW2 Unit 2: Fun in Class

## Official Scope

| Field | Content |
|---|---|
| Theme | Fun in Class |
| Goals | Say what people are doing; say when I use classroom objects; show where things are. |
| Vocabulary 1 | coloring, counting, cutting, drawing, erasing, gluing, talking |
| Vocabulary 2 | glue, a marker, a notebook, a paintbrush, scissors |
| Grammar 1 | Present progressive, first person plural (What are you doing? We're counting crayons.) |
| Grammar 2 | Are there...? / There are / There aren't (Are there any markers on the desk? No, there aren't.) |
| Song / Sounds | Song: Our Classroom · /k/ as in counting |
| Reading | Paper Art |
| Value | Be neat. |
