# EOW1 Unit 2: My World

## Official Scope

| Field | Content |
|---|---|
| Theme | My World |
| Goals | Name things in nature; talk about natural things; ask where things are. |
| Vocabulary 1 | bird, butterfly, river, rock, sky, sun, tree |
| Vocabulary 2 | bush, cloud, flower, moon, star |
| Vocabulary Strategy | Ending `-s` |
| Grammar 1 | `to be`: is, are |
| Grammar 2 | Where and in or on |
| Reading | Rainbows |
| Reading Strategy | Use visuals to support comprehension |
| Value | Enjoy nature. |
