# EOW1 Unit 4: My House

## Official Scope

| Field | Content |
|---|---|
| Theme | My House |
| Goals | Talk about things in a house; say where things are; talk about actions. |
| Vocabulary 1 | bathroom, bedroom, dining room, kitchen, lamp, living room, sofa |
| Vocabulary 2 | cleaning, cooking, eating, sleeping, taking a bath, watching TV |
| Vocabulary Strategy | Classifying and categorizing; base words and suffix `-ing` |
| Grammar 1 | Yes / No questions with `Is there...?` |
| Grammar 2 | Present progressive: He / She is + verb-ing |
| Reading | Houses Are Different |
| Reading Strategy | Make connections to personal experience |
| Value | Be neat. |
